Qu’est-ce que le HTML ?

Le HTML est un langage de balisage. Il permet de créer et structurer des pages Web. Par définition, toutes les pages Web utilisent le format HTML, le langage hypertexte. Ces pages HTML sont souvent générées en tout ou partie automatiquement.

Voici nos conseils pour réussir vos pages HTML.

  1. Fermez toujours vos balises HTML.
  2. Utilisez des feuilles de style chaque fois que possible.
  3. Utilisez un validateur HTML.
  4. Utilisez judicieusement les commentaires HTML
  5. Incorporez des images en HTML.
  6. Ecrivez un code propre HTML
  7. Toujours séparer le code CSS du code HTML

Ces conseils sont donnés à titre indicatif, il peut y en avoir d’autres.

Il existe 113 balises HTML

Voici dans ce petit tableau un rapide aperçu des balises existantes.

Quelques balises HTML à retenir
h1 à h6 pour créer des titres
p pour créer des paragraphes
table tr et td pour créer des tableaux
img pour insérer des images
title pour donner un titre au document html
head pour y insérer les balises métadonnées.

Qu’est-ce que le CSS ?

A quoi sert le CSS ou encore les feuilles de style en cascade ?

  • A séparer le contenu de la présentation.
  • Il organise les différentes parties d’un site
  • Mettre en forme des documents web.
  • Le CSS nous permet de gagner du temps.
